Upcoming Events

Wiki Wars

Wiki Wars is a skilful and smart game of navigating through several pages of Wikipedia on a desktop without the use of a keyboard. It involves reaching the target Wikipedia page from an already given Wikipedia page in the limited time and with the least number of clicks. The number of players for the game is 2. Wiki Wars will be conducted in three mega rounds.
